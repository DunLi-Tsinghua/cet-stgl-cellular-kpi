\section{Data Availability and Reproducibility}

The raw dataset is a private cellular operational record and cannot be released as a public benchmark in its current form. It contains business-sensitive KPI and alarm information and may reveal network-operation characteristics. Public release would require enterprise approval, anonymization review and possibly synthetic or schema-only alternatives.

The source code, sanitized data schema, synthetic toy data, and table/figure artifacts are available at \url{https://github.com/DunLi-Tsinghua/cet-stgl-cellular-kpi}. The repository does not include the raw \texttt{510957.csv} file. Exact metric reproduction requires authorized access to the private raw data and should follow the chronological split, train-only threshold fitting, train-only event influence estimation, and inferred-graph construction described in this paper.

The reproducible components of this project are the experimental protocol, code skeleton, feature-construction logic, model-status reporting and table/figure generation scripts. If allowed by the data owner, an anonymized schema, toy example or synthetic panel can be released in future work to demonstrate the pipeline without exposing operational records. Until then, all reported metrics should be interpreted as results on the private \texttt{510957.csv} slice, and not as a public benchmark leaderboard.
